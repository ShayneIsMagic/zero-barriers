# Contact Form Setup - Formspree

## Recommended Solution: Formspree

**Why Formspree:**
- ✅ Very reputable (used by thousands of sites)
- ✅ Free tier: 50 submissions/month
- ✅ Works perfectly with static sites (Cloudflare Pages, GitHub Pages)
- ✅ No backend code needed
- ✅ Easy setup
- ✅ Spam protection built-in
- ✅ Email notifications

## Setup Steps

1. **Create Account**: Go to https://formspree.io and sign up (free)

2. **Get Form Endpoint**: 
   - Create a new form in Formspree dashboard
   - You'll get a form endpoint like: `https://formspree.io/f/YOUR_FORM_ID`
   - Copy this endpoint

3. **Update Contact Form**:
   - The form is already configured to use Formspree
   - Just replace `YOUR_FORMSPREE_ENDPOINT` in the contact form with your actual endpoint

4. **Configure Email**:
   - In Formspree dashboard, set your email to receive submissions
   - Default: sk@zerobarriers.io

## Alternative Options

### Web3Forms (Also Free)
- Free tier: Unlimited submissions
- Website: https://web3forms.com
- Similar setup to Formspree
- Just need to get an access key

### EmailJS (Client-Side Only)
- Free tier: 200 emails/month
- Website: https://www.emailjs.com
- Requires client-side JavaScript
- More complex setup

## Recommendation

**Use Formspree** - It's the most reputable, widely-used, and reliable option for static sites.
